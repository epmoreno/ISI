# PROYECTO INGENIERÍA DE SISTEMAS DE INFORMACIÓN

## Entorno 📦

+ React + NodeJS
+ FastApi
+ mysql
+ phpMyAdmin
+ mongodb
+ mongo-express
  
## Instalación 🚀

1. Clonar Repositorio.
2. Copiar .env.example a .env y rellenar con sus datos.
3. Ejecutar docker compose up --build

## Esquema de Nateo 🤝

+ Frontend: localhost:8001 -> Docker:5173
+ API: localhost:8000 -> Docker:8000
+ Mongo-Express: localhost:8081 -> Docker:8081
+ phpMyAdmin: localhost:8082 ->  Docker:80
